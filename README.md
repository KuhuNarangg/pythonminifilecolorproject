# Python Image Processing Miniproject

A collection of simple Python scripts for image processing using OpenCV.

## Features
- Convert images to grayscale (`cv.py`, `main.py`).
- Apply filters, blurring, and edge detection (`hey.py`).
- Batch process images in the `images` folder.

## Getting Started
1. Clone the repository.
2. Install dependencies: `pip install opencv-python`.
3. Place images in the `images` folder.
4. Run scripts: `python cv.py` or `python hey.py`.

## Contents
- `cv.py`: Batch grayscale conversion using `cv2.cvtColor`.
- `hey.py`: Advanced image processing (resize, blur, edge detection).
- `main.py`: Simple version of the grayscale conversion.
